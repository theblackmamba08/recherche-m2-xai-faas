## Objectif

Présenter de façon générale les séries chronologiques (vocabulaire, composantes, modélisation déterministe) comme socle préliminaire au sujet de mémoire. L'objectif était encore flou à ce stade : pas encore de lien explicite établi avec XAI ni avec FaaS.

## Points clés présentés

1. **Introduction** : définition d'une série chronologique `(X_t)_{t∈T}`, types (temps discret / continu), domaines d'application (économie, médecine, démographie).
2. **Composantes** : tendance (T_t), saisonnalité (S_t), résidu/bruit (R_t), phénomènes accidentels, cycle.
3. **Objectifs de l'étude** : décrire, contrôler, prévoir, détecter les ruptures, analyser les relations.
4. **Étapes d'une étude** : correction des données → observation graphique → modélisation → analyse par composantes → diagnostic → prédiction.
5. **Modélisation déterministe** : modèle additif `X_t = T_t + S_t + R_t`, modèle multiplicatif `X_t = T_t × S_t × R_t`, modèles mixtes, choix du modèle.
6. **Analyse de la tendance** : régression linéaire par moindres carrés, points médians, ajustements non linéaires, moyennes mobiles.
7. **Décomposition** : lissage → estimation tendance → estimation saisonnalité → analyse résidus → prévision.
